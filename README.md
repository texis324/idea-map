# ⚡ IDEA MAP

Xmind ＋ 横から殴ってくるAI。発想を **足し算（手書き）** から **掛け算（AI噛ませ）** に変えるマインドマップ。単一HTML・依存ゼロ。

## なにこれ
普通のマインドマップは手で書いた分しか枝が増えない＝足し算。自分の脳みそ1台が上限。
このツールはノードに **⚡** を押すとAIが別角度に枝を生やし、**🎲** で離れた2ノードを強引に交差させる。1ノードの情報量 × AIの発散 × 自分が選ぶ回数で、アイデア空間が指数的に開く。

## 使い方
1. `index.html` をブラウザで開く（スマホ可）
2. 右上の **⚙** に Anthropic APIキーを入れる（localStorageにだけ保存・外に出ない）
3. ノードを選んで **⚡展開**（拡散／逆張り／異分野比喩／極端化／兄弟と交配）or **🎲強制連想**
4. 料金は右上に **¥** 表示（Haikuなら1展開0.1円未満）

AIを **OFF** にすればただの軽い手書きマインドマップとしても使える。

### 操作
| | PC | スマホ |
|---|---|---|
| ノード追加 | 空白ダブルクリック | 空白2回タップ |
| 編集 | ノードをダブルクリック | ノード2回タップ |
| 移動 / パン | ドラッグ / 背景ドラッグ | 1本指 |
| ズーム | ホイール | 2本指ピンチ |
| ショートカット | Tab=子・Enter=兄弟・E=展開・Del=削除 | 選択ノードのリング ⚡＋✕ |

## 技術
- 単一HTML（Vanilla JS + SVG）。ビルド不要・サーバ不要
- AIはブラウザ直 fetch → Anthropic Messages API（`anthropic-dangerous-direct-browser-access` ヘッダ）
- 状態は localStorage 永続。JSON 書き出し／読み込み対応
